Real Estate Analytics Project (Chennai Market)
Project Overview

This project analyzes residential real estate data to derive city-level and micro-market insights using structured SQL queries. The objective is to evaluate market size, pricing behavior, supply distribution, and investment attractiveness across locations in a data-driven manner.

Dataset

The analysis is based on residential property listing data containing attributes such as:

Location

Price

Area

Number of Bedrooms

Amenities and Infrastructure Indicators

Key KPIs

Metropolitan Real Estate Market Size & Ranking

City-wise Share of Property Listings

Location-wise Price Index (Median PPSF)

Property Overpricing Ratio Index (ORI)

Location Investment Attractiveness Score (LIAS)

Bedroom-wise Listing Distribution

Location-wise Average Property Size

Top 5 Locations by Average Price per Sq Ft

Tools & Technologies

SQL (Window Functions, CTEs, Aggregations)

Excel / BI Tool (for visualization)

Microsoft Word (KPI Report)

Project Structure

KPI_Report.docx – Business-focused KPI explanations and insights
